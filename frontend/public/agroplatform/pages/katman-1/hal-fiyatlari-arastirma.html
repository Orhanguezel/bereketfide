Hal Fiyatları — Veri Kaynakları Analizi

Türkiye'deki hal fiyat verilerine erişim yolları, mevcut API'ler, veri kalitesi ve AgroPlatform için teknik entegrasyon stratejisi.
Stratejik Sonuç: Veri bankasını sıfırdan oluşturmaya gerek yoktur. Birden fazla büyükşehir belediyesi açık ve ücretsiz API sunmaktadır. Bizim görevimiz bu verileri tek formatta birleştirip akıllı UX ve analitik araçlarla sunmaktır.
Resmi API Kaynakları Hazır & Ücretsiz
ANA KAYNAK

İstanbul İBB — Hal Ürünleri API

Swagger dökümantasyonlu REST API. JSON formatında, kayıt gerektirmeyen ve en kapsamlı veri setine sahip kaynaktır.

halfiyatlaripublicdata.ibb.gov.tr/swagger/ui/index
Kritik Endpoint'ler:
  • GET Ürün kategorileri ve ID eşleşmeleri
  • GET Günlük birim fiyatlar (Min/Max/Ort)
  • GET Tarih bazlı geçmiş veri sorgulama
EGE BÖLGESİ

İzmir Büyükşehir — Açık API

Günlük sebze ve meyve hal fiyatları. JSON formatında, tarih parametresi ile sorgulanabilen stabil bir servistir.

openapi.izmir.bel.tr/api/ibb/halfiyatlari/...
DİĞER KAYNAKLAR
  • Balıkesir: REST API (Açık Veri Portalı)
  • Bursa: CSV/JSON (Açık Yeşil Portalı)
  • Konya: Günlük PDF/CSV Dökümleri

Teknik İş Akışı ve Veri Mimarisi

Veri Toplama Akışı (ETL)

Cron Job (Her gün 06:15)
  [FETCH] --> IBB & Izmir API
  [PARSE] --> Normalize (Havuç <-> Havuc)
  [STORE] --> market_prices Table
  [NOTIFY] --> Threshold Alarms triggered
  [STATS] --> Update Trend Graphics

Veritabanı Şeması (PostgreSQL)

TABLE market_prices (
  id BIGSERIAL PRIMARY KEY,
  product_name VARCHAR(100),
  city_name VARCHAR(50),
  min_price DECIMAL(10,2),
  max_price DECIMAL(10,2),
  avg_price DECIMAL(10,2),
  price_date DATE INDEX,
  source_api VARCHAR(50)
);
